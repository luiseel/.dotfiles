# .dotfiles

These are just my dotfiles. Feel free to use them.

## Install

Clone the repository.

```sh
git clone https://github.com/luiseel/.dotfiles ~/.config/.dotfiles
```

`cd` to cloned repo.

```sh
cd ~/.config/.dotfiles
```

Run stow to install the dotfiles.

```sh
stow --no-folding -t ~ -R .
```
